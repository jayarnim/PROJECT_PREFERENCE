### 주요 내용

- **경제지수와 트렌드 상관관계 분석 및 예측**
  - 시대별 트렌드 요약 및 시각화
  
  - 2000년대 이후 경제지수와 대중문화 간 상관관계 분석
    - 경제지수 : GDP, 인플레이션율, 주가 등
    - 대중문화 : 음악, 드라마/영화, 베스트셀러 등
  
  - 후년도 대중문화 분야별 키워드 도출
    - 정부에서 예측한 후년도 경제지수에 기초함

- **상권 분석 및 키워드 정의**
  - 서울시내 상권별 평당 임대가격과 주요품목, 상품 가격 간 상관관계 분석
  
  - 서울시내 상권별 키워드 도출
    - ex) 성수동 : `공장` `재생`
  
- **웹 페이지 `TRANDREPORT` 설계**
